class omni.isaac.orbit.objects.articulated.ArticulatedObject[source]#

Bases: object

Class for handling articulated objects.

This class wraps around ArticulationView and RigidPrimView classes from Isaac Sim to support the following:

  • Configuring the object using a single dataclass (struct).

  • Applying settings to the articulated object from the configuration class.

  • Handling different rigid body views inside the articulated object.

  • Storing data related to the articulated object.

spawn(prim_path: str, translation: Sequence[float] | None = None, orientation: Sequence[float] | None = None)[source]#

Spawn an articulated object into the stage (loaded from its USD file).

Note

If inputs translation or orientation are not None, then they override the initial root state specified through the configuration class at spawning.

Parameters:

  • prim_path (str) – The prim path for spawning object at.

  • translation (Sequence[float], optional) – The local position of prim from its parent. Defaults to None.

  • orientation (Sequence[float], optional) – The local rotation (as quaternion (w, x, y, z) of the prim from its parent. Defaults to None.

initialize(prim_paths_expr: str | None = None)[source]#

Initializes the PhysX handles and internal buffers.

Note

PhysX handles are only enabled once the simulator starts playing. Hence, this function needs to be called whenever the simulator “plays” from a “stop” state.

Parameters:

prim_paths_expr (Optional[str], optional) – The prim path expression for the prims. Defaults to None.

Raises:

RuntimeError – When input prim_paths_expr is None, the method defaults to using the last prim path set when calling the spawn() function. In case, the object was not spawned and no valid prim_paths_expr is provided, the function throws an error.

Parameters:

update_buffers(dt: float)[source]#

Update the internal buffers.

Parameters:

dt (float) –

The amount of time passed from last update_buffers call.

This is used to compute numerical derivatives of quantities such as joint accelerations which are not provided by the simulator.

get_random_dof_state(env_ids: Sequence[int] | None = None, lower: float = 0.5, upper: float = 1.5) → Tuple[torch.Tensor, torch.Tensor][source]#

Returns randomly sampled DOF state (position and velocity) of actor.

Currently, the following sampling is supported:

  • DOF positions:

    • uniform sampling between (lower, upper) times the default DOF position.

  • DOF velocities:

    • zero.

Parameters:

  • env_ids (Optional[Sequence[int]], optional) – Environment indices. Defaults to None (all environment indices).

  • lower (float, optional) – Minimum value for uniform sampling. Defaults to 0.5.

  • upper (float, optional) – Maximum value for uniform sampling. Defaults to 1.5.

Returns:

The sampled DOF position and velocity of the actor.

Each tensor has shape: (len(env_ids), 1).

Return type:

Tuple[torch.Tensor, torch.Tensor]
