Base class for robots.

The robot class is strictly treated as the physical articulated system which typically runs at a high frequency (most commonly 1000 Hz).

This class wraps around ArticulationView class from Isaac Sim to support the following: * Configuring using a single dataclass (struct). * Handling different rigid body views inside the robot. * Handling different actuator groups and models for the robot.
